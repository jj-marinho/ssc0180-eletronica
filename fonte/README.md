## Objetivo do Projeto
Desenvolver uma fonte elétrica capaz de fornecer energia para pequenos projetos de eletrônica, como projetos de Arduino.

## Critérios da Fonte
- Componentes elétricos simples
- Tensão Ajustável através de um potenciômetro (3v - 12v)
- Corrente de 100mA
